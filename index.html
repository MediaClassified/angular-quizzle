Getting Started

  1. Install Quizzle via bower or download the latest zip. bower install angular-quizzle --save
  2. Include AngularJS in your project.
    3. <script src="/path/to/angular/angular.min.js" ></script>
  3. Include Quizzle and its dependencies.
    4. <script src="/path/to/angular-animate/angular-animate.min.js" ></script>
    <script src="/path/to/angular-quizzle/dist/quizzle.js" ></script>
    <link rel="stylesheet" href="/path/to/angular-quizzle/dist/quizzle.css" />
  4. Load the module in your application by including it as a dependency.
    5. angular.module('app', ['quizzle']);
  5. Generate a quiz or create your own. If you create your own quiz, be sure to follow the guidelines outlined in the documentation.
  6. Include the quizzle directive in your HTML with your quiz object.
    7. <quizzle quiz="quiz" repeat="true" ></quizzle>

For more instructions on how to use this directive, please refer to the documentation.
